# Calculator App (Flutter)

Yeh ek simple Calculator application hai jise maine **Flutter** aur **Dart** ka use karke banaya hai.

## Features
- Basic mathematical operations (Addition, Subtraction, Multiplication, Division, Percentage).
- Dark mode theme design.
- Clean and responsive user interface.

## Technologies Used
- **Language:** Dart
- **Framework:** Flutter

## How to Run
1. Ensure you have the Flutter SDK installed on your machine.
2. Clone this repository to your local machine.
3. Open the project in your code editor (like VS Code or Android Studio).
4. Run `flutter pub get` to install dependencies.
5. Execute `flutter run` to launch the app on your emulator or connected device.
